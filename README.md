# Miracart Landing

Промо-лендинг для сервиса подбора, закупки и доставки материалов для ремонта.
Проект собран как быстрый маркетинговый сайт с островной интерактивностью на Vue: основная страница остаётся статичной и лёгкой, а формы и модальные сценарии подключаются точечно.

## Что здесь важно

- Статический лендинг на `Vite` + `Handlebars`
- Островная интерактивность на `Vue 3`
- `TypeScript` для island-компонентов и composables
- Модульная SCSS-архитектура
- Генерация SVG-спрайта из локальных иконок
- Автоконвертация изображений в `WebP`

## Стек

## Архитектура

Проект разделён на две части:

1. Статическая оболочка лендинга рендерится через `index.html` и Handlebars partials.
2. Интерактивные блоки монтируются как Vue islands через `data-vue-island`.

Это позволяет держать страницу быстрой, не превращая весь лендинг в SPA.

## Основные директории

```text
src/
  components/   Vue-компоненты форм и модалки
  islands/      точки монтирования island-компонентов
  composables/  переиспользуемая логика форм, модалок и маски телефона
  styles/       SCSS-архитектура и стили секций
  assets/       импортируемые ресурсы

partials/
  layout/       head, header, footer, scripts, svg sprite
  sections/     секции лендинга

public/
  images/       статические изображения
  fonts/        локальные шрифты

scripts/
  generate-svg-sprite.js
  convert-images.js
```
